# SmartPark
Projekt na zaliczenie przedmiotu "Wstęp do programowania obiektowego" Przygotowany przez studentów identyfikujących się numerami albumu: 79591 oraz 79900 


SmartPark to system przeznaczony do zarządzania parkingiem, który znacząco ułatwi i usprawni codzienną obsługę obiektu. Dzięki jego zastosowaniu właściciele parkingów będą mogli ograniczyć liczbę pomyłek, podnieść jakość obsługi kierowców oraz efektywniej wykorzystywać dostępne miejsca postojowe.

Lista funkcjonalności:
1. Rejestracja pojazdu
2. Przydzielanie miejsca
3. Zwalnianie miejsca
4. Liczenie opłat
5. Sprawdzanie dostępności miejsc
6. Raport dzienny
7. Kara za przekroczenie czasu
8. Historia parkowań
9. Statystyki obciążenia miejsc
